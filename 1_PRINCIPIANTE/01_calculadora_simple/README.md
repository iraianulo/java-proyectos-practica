# 🧮 Ejercicio: Calculadora Simple en Java

## 📌 Enunciado

Crea una aplicación en Java que funcione como una calculadora básica por consola.

El programa debe permitir al usuario introducir dos números y un operador, y devolver el resultado de la operación.

---

## 🎯 Requisitos

### 🔢 Entrada de datos

El usuario debe introducir:

* Un primer número (decimal permitido)
* Un operador: `+`, `-`, `*`, `/`, `^`
* Un segundo número

---

### ⚙️ Operaciones

El programa debe soportar:

* Suma (`+`)
* Resta (`-`)
* Multiplicación (`*`)
* División (`/`)
* Potencia (`^`)

---

### ⚠️ Validaciones

* No permitir división entre 0
* Detectar operadores no válidos
* Mostrar mensajes de error adecuados

---

## 🖥️ Salida

* Mostrar el resultado de la operación si es válida
* En caso de error, mostrar mensaje y no imprimir resultado

---

## 🧠 Pistas

* Usa `Scanner` para leer datos
* Usa `switch` para gestionar operadores
* Puedes usar `Math.pow()` para la potencia
* Usa un boolean para controlar si la operación es válida

---

## 🚀 Extra (opcional)

* Permitir múltiples operaciones (bucle)
* Añadir más operadores (raíz, módulo, etc.)
* Validar mejor la entrada del usuario

---

## ✅ Objetivo

Practicar estructuras básicas de Java:

* Entrada de datos
* Condicionales
* Switch
* Control de errores simple
